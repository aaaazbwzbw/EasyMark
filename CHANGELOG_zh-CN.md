# 更新日志

EasyMark 的所有重要变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [1.0.2] - 2025-12-27

### 新增
- **SAM2**
  - 支持 `Ctrl + Shift + 点击` 连续追加提示点，用于在当前分割结果基础上持续细化
- **Python 环境管理**
  - 支持为 Python 插件绑定外部虚拟环境
- **角标提示**
  - Python 环境角标状态持久化，不再每次切换页面都检查

### 调整
- 插件安装/更新改为覆盖拷贝（不再删除整个插件目录），避免用户已下载权重被误删

### 修复
- 修复 Electron IPC 传递 Vue Proxy 对象导致的结构化克隆报错
- 优化 SAM2 mask 转 polygon 的后处理，降低毛边/锯齿
- 修复导出时选择冲突类别：提示冲突后 checkbox 仍可能被勾选的问题
- 修复内置 `dataset-common` 插件可执行文件未正确构建/打包的问题
- 降低页面切换导致的 `/api/python/plugins-deps-summary` 频繁检查

## [1.0.0] - 2025-12-02

> **EasyMark** 是 **YoloMarkFlow** 完全重构后的第一个版本。
> 
> 相比原项目，EasyMark 采用了全新的技术架构（Electron + Vue 3 + Go），
> 提供更现代的 UI、更强大的插件系统和更完善的数据集管理能力。
> 
> 重构后使用 Go 后端处理文件操作，大数据量场景下性能大幅提升。

### 新增
- **标注功能**
  - 矩形框标注（目标检测）
  - 多边形标注（图像分割）
  - 关键点标注（姿态估计）
  - 自动保存与手动保存模式

- **AI 辅助**
  - SAM2 交互式分割插件
  - YOLO 自动检测插件
  - 训练模型自动导入推理

- **数据集管理**
  - 支持 COCO、YOLO、VOC 格式导入
  - 支持 COCO、YOLO、VOC 格式导出
  - 数据集版本管理与回溯
  - 跨项目类别选择导出

- **模型训练**
  - Ultralytics YOLOv8/v11 训练支持
  - 检测、分割、姿态估计任务
  - 训练历史与指标可视化

- **插件系统**
  - 数据集导入导出插件
  - Python 推理插件
  - 训练插件与进度追踪

- **用户体验**
  - 深色/浅色主题
  - 键盘快捷键
  - 多语言支持（中文、英文）
