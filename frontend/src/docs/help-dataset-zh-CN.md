# 数据集页使用指南

数据集页用于管理标注数据的版本和导出。

---

## 1. 进入数据集页面

完成标注后，点击左侧栏的**数据集管理**按钮进入数据集页面。

---

## 2. 同步版本

鼠标悬浮一个项目后，会显示**同步按钮**。点击同步按钮即可更新或创建该项目的一个版本。

> 版本是项目标注状态的快照，方便回溯和管理。

---

## 3. 导出数据集

### 3.1 选择导出内容

选中不同项目、不同版本内的类别：

- **支持**：跨项目、跨版本多选
- **不支持**：同项目内不同版本的相同类别

### 3.2 执行导出

1. 选中类别后，右上角会出现**导出数据集**按钮
2. 点击后配置：
   - 导出路径
   - 导出格式（YOLO、VOC、COCO 等）
   - 数据集划分比例（训练集/验证集/测试集）
3. 点击**开始导出**

---

## 4. 版本管理

右键左侧树结构中的某个版本，可进行以下操作：

| 操作 | 说明 |
|------|------|
| **删除** | 删除该版本 |
| **回溯** | 二次确认后，将项目恢复到此版本的状态 |

> **回溯说明**：回溯会将项目的标注状态恢复到该版本创建时的状态，当前未保存的修改会丢失。

---

## 5. 与其他页面的关系

- **项目页**：数据集版本是项目标注状态的快照
- **训练页**：训练任务的数据源来自数据集版本
- **插件页**：安装新的导出插件可扩展导出格式
