# 训练页使用指南

训练页用于创建和管理模型训练任务。

---

## 1. 进入训练页面

点击左侧栏的**训练管理**按钮进入训练页面。

---

## 2. 前置条件

开始训练前请确保：

- 相关插件已就绪
- 插件依赖已安装（见 Python 环境管理章节）

---

## 3. 创建训练集

### 3.1 创建步骤

1. 点击**创建训练集**按钮
2. 选择版本中的类别
3. 输入训练集名称，点击确定

### 3.2 类别选择规则

| 支持 | 不支持 |
|------|--------|
| 同项目内不同版本的相同类别 | 跨项目选择类别 |

### 3.3 管理训练集

已保存的训练集可在右侧进行：

- **删除**：移除训练集
- **更新**：将训练集中的类别替换为当前选中的类别

---

## 4. 新建训练任务

点击**新建训练任务**后，配置以下参数：

| 配置项 | 说明 |
|--------|------|
| 训练名称 | 任务的标识名称 |
| 数据来源 | 选择训练集或从目录选择数据集 |
| 数据划分 | 配置训练集、验证集比例 |
| 训练类型 | 矩形框、多边形、关键点 |
| 训练插件 | 选择支持的训练插件 |
| 模型与版本 | 选择合适的模型和版本 |
| 训练参数 | 配置超参数（epochs、batch size 等） |

配置完成后点击确定开始训练。

---

## 5. 训练结果

训练完成后：

- 显示在右侧**训练历史**中
- 可直接删除或打开目录查看模型和指标
- 训练好的模型会**自动导入**到 YOLO 推理插件中，可直接使用

---

## 6. 与其他页面的关系

- **数据集页**：训练集的数据来源
- **推理插件**：训练完成的模型自动可用
- **插件页**：扩展训练插件和模型支持
