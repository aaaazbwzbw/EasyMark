# 项目页使用指南

项目页是进行图像标注的核心工作区域。

---

## 1. 创建/选择项目

1. 打开程序后，点击左侧的**项目管理器**
2. 点击右上角的**项目菜单**选择或创建一个项目

---

## 2. 导入图片或数据集

创建项目后，可以导入图片或导入已有数据集。

### 支持的数据集格式

- YOLO
- VOC
- COCO

### 导入方式

| 方式 | 说明 |
|------|------|
| **复制** | 将文件复制到项目目录 |
| **硬链接** | 创建硬链接节省磁盘空间 |
| **外部索引** | 仅索引不复制，保留原始位置 |

---

## 3. 配置标注类别

在右侧面板切换需要标注的类型：

| 类型 | 说明 |
|------|------|
| **矩形框** | 目标检测常用 |
| **多边形** | 实例分割/语义分割 |
| **关键点** | 姿态估计 |

### 关键点特殊配置

关键点需要额外配置：
1. 设置各点的语义（如：鼻子、左眼、右眼等）
2. 绑定一个矩形框类别

> **说明**：绑定后，推理时同矩形框类别的检测结果会自动使用绑定的关键点语义。

---

## 4. 开始标注

### 4.1 选择图片

点击左侧图片列表中的图片，中间画布会显示该图片。

### 4.2 选择类别

- 可同时选中矩形框和关键点类别
- 多边形类别**不能**与其他类别同时选中

### 4.3 绘制标注

| 标注类型 | 操作方式 |
|----------|----------|
| **矩形框** | 按住鼠标左键拖拽绘制 |
| **多边形** | 按住 `Alt` + 鼠标左键依次点击顶点 |
| **关键点** | 按住 `Alt` + 鼠标左键按顺序点击 |

### 4.4 取消绘制

多边形和关键点在未完成绘制前，右键点击空白区域可取消。

---

## 5. 保存标注

- 点击**保存按钮**手动保存
- 或开启底部栏的**自动保存**功能

---

## 6. AI 辅助标注

点击底部栏的**插件按钮**打开插件小窗。

### 内置辅助插件

| 插件 | 使用方式 |
|------|----------|
| **SAM2** | 加载模型后，选中多边形或矩形框类别，按住 `Shift` 点击目标进行分割推理 |
| **YOLO** | 选中模型后，切换图片会自动推理 |

> **注意**：推理渲染的结果需要**手动保存**，自动保存不会保存推理结果。

---

## 7. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` | 保存当前图片标注 |
| `Ctrl + Shift + S` | 保存为负样本 |
| `← / →` | 上一张 / 下一张图片 |
| `Ctrl + ← / →` | 跳转上/下一张未标注图片 |
| `Backspace` | 删除选中的标注 |
| `V` | 切换关键点可见性 |
| `Ctrl + 0` | 重置画布视图 |
